# Android Store / PWA Builder Listing Text

## Short Description

Fullscreen Plant 076 3D warehouse simulation with mobile controls and WebXR/VR support.

## Full Description

Plant 076 is an interactive 3D warehouse simulation built as an installable PWA. It launches in fullscreen landscape mode, includes an in-app fullscreen control, supports walk mode and mobile touch controls, and exposes WebXR/VR launch support for compatible Android browsers and headset browsers.

Use it to inspect the Plant 076 warehouse layout, production lines, racks, floor geometry, aisles, and operational spaces in a single offline-capable simulation.

## Capability Notes For PWA Builder

- Fullscreen display is declared in `manifest.json`.
- The app includes a visible fullscreen control in the top bar.
- WebXR/VR support is present through the Three.js VR button.
- The service worker caches the app shell for offline launch after first install.
